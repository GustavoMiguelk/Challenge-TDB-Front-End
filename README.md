## Descrição do Projeto

A **Solução do Bem** é uma Single Page Application (SPA) desenvolvida como projeto acadêmico para a disciplina de Front-End Design Engineering, em parceria com a ONG **Turma do Bem**, ONG que oferece atendimento odontológico gratuito para crianças, adolescentes em situação de vulnerabilidade social e mulheres que em decorrência de violência tiveram sua saúde bucal afetada, eles operam em todo o Brasil e em mais de 12 países.
O objetivo do projeto é criar uma presença digital moderna e acessível para a ONG, facilitando o acesso a informações sobre seus programas, formas de voluntariado, doações e contato. A aplicação foi desenvolvida com foco em componentização, responsividade e boa experiência do usuário, utilizando as tecnologias mais atuais do mercado front-end.

## Como Executar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/GustavoMiguelk/Challenge-TDB-Front-End.git

# 2. Entre na pasta do projeto
cd Challenge-TDB-Front-End

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev

# 5. Acesse no navegador
# http://localhost:5173
```

### Para rodar a API Java (necessário para Login e Dashboard):

```bash
# Na pasta do projeto Java
# Abra pelo IntelliJ IDEA: Maven → quarkus:dev
# A API ficará disponível em http://localhost:8080
```
